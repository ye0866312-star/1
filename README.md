# GitHub Skills Documentation

Welcome to the GitHub Skills repository!

## Setup Instructions

1. **Fork this repository** to your personal GitHub account.
2. **Clone your forked repository** to your local machine using:
   ```bash
   git clone https://github.com/<your-username>/1.git
   ```
3. **Navigate into the directory**:
   ```bash
   cd 1
   ```
4. **Create a new branch** for your changes:
   ```bash
   git checkout -b my-skills
   ```
5. **Make your changes** and add files as needed.
6. **Commit your changes** and push the branch:
   ```bash
   git push origin my-skills
   ```
7. **Open a pull request** in the original repository to submit your changes.

Happy coding!
